# Bubble Sort Visualizer

This project is a visual representation of the Bubble Sort algorithm using HTML, CSS, and JavaScript.

## What does this project do?

The user enters a list of numbers separated by commas, then a stickman character moves between the balls and sorts the numbers step by step using the Bubble Sort algorithm.

## Features

Enter custom numbers.

Visual Bubble Sort animation.

Stickman movement during sorting.

Random reshuffle and re-sort.

Displays algorithm complexity.

## Complexity

Time Complexity: O(n²)

Space Complexity: O(1)

## Technologies Used

HTML5

CSS3

JavaScript

## How to Run

Download or clone the project.

Open index.html in your browser.

Enter numbers separated by commas.
